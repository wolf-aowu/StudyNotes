# numpy

## 创建数组

```python
import numpy as np
# 创建一维数组
a = np.array([1, 2, 3, 4])
# 创建二维数组
b = np.array([[1, 2], [3, 4], [5, 6]])
print(a)
print(type(a))
print(b)
```

输出：

```python
[1 2 3 4]
<class 'numpy.ndarray'>
[[1 2]
 [3 4]
 [5 6]]
```

## 数学运算

``` python
import numpy as np
a = np.array([1, 2, 3, 4])
c = a * 2
print(c)
```

输出：

```python
[2 4 6 8]
```

如果是列表进行 * 2 运算，则会像下面这样复制一遍元素。

```python
[1, 2, 3, 4, 1, 2, 3, 4]
```
