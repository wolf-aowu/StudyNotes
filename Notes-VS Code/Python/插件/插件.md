# 好用插件

## 函数注释头

### koroFileHeader

优点：

1. 配置文档易上手
2. 支持函数外注释

缺点：

1. 无法识别 是否存在返回值 和 返回值类型
2. 光标移至函数名上不显示注释（外部注释是这样，内部注释没有试过）

#### 快捷键

函数注释：`Ctrl + Win + t`

函数外注释：将光标放在函数上方空白行或放在函数行

函数内注释：将光标放在函数内

文件头注释：`Ctrl + Win + i`

### autoDocstring - Python Docstring Generator

优点：

1. 可以识别 返回值类型 和 默认值
2. 光标移至函数名上能显示注释

#### 快捷键

函数注释：`Ctrl + Shift + 2`
